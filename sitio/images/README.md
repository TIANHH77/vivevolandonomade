# Imágenes del Sitio

Esta carpeta debe contener las imágenes optimizadas para el sitio web.

## Estructura de archivos necesarios:

### Logos
- `logo-blanco.png` - Logo en blanco para header y footer (fondo oscuro)
- `logo-color.png` - Logo en color para otros usos

### Hero
- `hero-salar.jpg` - Imagen principal del hero (Salar espejo atardecer)
  - Dimensiones recomendadas: 1920x1080px
  - Optimizada para web

### Sobre
- `paramotor-equipo.jpg` - Foto del paramotor rojo en hangar
  - Dimensiones recomendadas: 800x600px

### Servicios
- `vuelo-paramotor.jpg` - Vuelo en paramotor 360°
- `laguna-gomez.jpg` - Vista de la Laguna de Gómez
- `fauna-local.jpg` - Carancos o fauna local
  - Dimensiones recomendadas: 600x400px cada una

### Galería
- `gallery-1.jpg` - Salar de litio aéreo
- `gallery-2.jpg` - Casa abandonada pampa
- `gallery-3.jpg` - Paramotor vuelo 360
- `gallery-4.jpg` - Carancos fauna local
- `gallery-5.jpg` - Laguna altiplánica
- `gallery-6.jpg` - Flamingos en vuelo
- `gallery-7.jpg` - Salar espejo atardecer
- `gallery-8.jpg` - Vehículo nómada
  - Dimensiones recomendadas: 800x800px (cuadradas)

## Optimización

Para optimizar las imágenes antes de subirlas:

1. Redimensionar a las dimensiones recomendadas
2. Exportar en formato JPG con calidad 80-85%
3. Usar herramientas como TinyPNG o Squoosh para comprimir sin perder calidad
4. Los logos deben ser PNG con transparencia

## Próximos pasos

1. Renombrar tus fotos según estos nombres
2. Optimizarlas para web
3. Subirlas a esta carpeta
4. El sitio las cargará automáticamente
